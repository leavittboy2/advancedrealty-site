# RM Unit Availability Plugin

## Description
This WordPress plugin pulls unit availability data from a Rent Manager database using the UA API and displays it on a WordPress site using shortcodes.

## Installation
1. Clone or download this repository.
2. Place it in your WordPress "plugins" directory.
3. Activate the plugin in the WordPress admin under “Plugins”.

## Dependencies
1. [Advanced Custom Fields](https://www.advancedcustomfields.com/)

## Usage
After activation, configure the plugin settings. Shortcodes provided by this plugin can be added to WordPress pages or posts to display availability data.

## Development
1. Install and activate all required dependencies.
2. Use typical WordPress plugin development practices for modifications.
3. Test thoroughly before committing changes.
